# Check Is Number

Two approaches for validating whether user input is a number. `num_input.py` uses `str.isdigit()` for integer-only validation, while `num_input_2.py` uses `float()` conversion to accept both integers and decimals, with an interactive loop.

## Usage

```bash
# Integer-only validation
python num_input.py

# Integer and float validation (interactive, type 'exit' to quit)
python num_input_2.py
```
