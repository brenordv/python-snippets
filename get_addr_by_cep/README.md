# Get Address by CEP

Five progressive examples of looking up a Brazilian address from a CEP (postal code) using the [ViaCEP](https://viacep.com.br/) public API. Each script builds on the previous one, adding status-code validation, user input, and increasingly thorough exception handling.

## Usage

```bash
# Simplest example -- hardcoded CEP, no error handling
python get_addr_01.py

# Interactive with full exception handling
python get_addr_05.py
```
