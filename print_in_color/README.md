# Print in Color

Three approaches to printing colored text in the terminal: raw ANSI escape codes (`printer.py`), the `sty` library (`print_sty.py`), and the `termcolor` library (`print_termcolor.py`).

## Usage

```bash
# ANSI escape codes (no dependencies)
python printer.py

# Using sty
python print_sty.py

# Using termcolor
python print_termcolor.py
```
