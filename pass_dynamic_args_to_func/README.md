# Pass Dynamic Args to Function

Three progressive demos showing how to pass arguments to Python functions: basic positional/keyword args, a verbose wrapper that checks every combination manually, and a clean approach using `**kwargs` dictionary unpacking.

## Usage

```bash
# Demo 0: positional and keyword arguments
python passing_args_to_func_00.py

# Demo 1: wrapper function -- the hard way
python passing_args_to_func_01.py

# Demo 2: wrapper function -- the easy way with **kwargs
python passing_args_to_func_02.py
```
