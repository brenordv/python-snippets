# OpenAI Generate Image and Resize

Generate images using the OpenAI DALL-E API (supports DALL-E 2 and DALL-E 3), download the result, and create multiple resized variants using Pillow. Includes input validation for model parameters, prompt length, and configuration options.

## Usage

```bash
# Set your API key
export OPENAI_API_KEY="sk-..."

# Run the demo (generates a corgi in space and creates resized copies)
python openai_gen_image_and_resize.py
```
